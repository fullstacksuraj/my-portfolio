HTML5 Developer Notes

HTML Boilerplate

HTML Boilerplate ek basic structure hota hai jo har HTML webpage banane ke liye use kiya jata hai.

  • <!DOCTYPE html> - Browser ko batata hai ki document HTML5 hai.
  • <html> - Pure webpage ka root element hota hai.
  • <head> - Isme webpage ki information hoti hai.
  • <body> - Visible webpage content yahan likha jata hai. 

<!DOCTYPE html>
<html lang="en">

<head>
  <title>Document</title>
</head>

<body>

</body>

</html>

Heading Tags

HTML me headings webpage structure ke liye use hoti hain.

  • <h1> - Sabse important heading.
  • <h2> - Secondary heading.
  • <h3> - Subheading ke liye use hota hai. 

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>

Paragraph Tag

Paragraph tag webpage me text content likhne ke liye use hota hai.

  • <p> - Paragraph create karta hai.
  • Browser automatically spacing deta hai.

<p>
This is a paragraph.
</p>

Anchor Tag

Anchor tag links create karne ke liye use hota hai.

  • <a> - Link create karta hai.
  • href - Destination URL specify karta hai. 

<a href="https://google.com">
Google
</a>

Image Tag

Images webpage par show karne ke liye img tag use hota hai.

  • src - Image path define karta hai.
  • alt - Alternative text show karta hai. 

<img src="image.jpg"
alt="image">

Lists

HTML me multiple items ko organize karne ke liye lists use ki jati hain. Mainly do types ki lists hoti hain: Unordered List aur Ordered List.

  • <ul> - Unordered list create karta hai.
  • <ol> - Ordered (numbered) list create karta hai.
  • <li> - List item define karta hai.
  • Lists menus, navigation aur grouped content dikhane ke liye use hoti hain.

HTML lists webpage content ko clean aur organized banati hain. 

 
<ul> 
  <li>
    HTML
    </li> 
    <li>
    CSS
    </li> 
</ul> 
<ol> 
  <li>
    JavaScript
  </li> 
  <li>
    React
  </li> 
</ol>

Tables

HTML me data ko rows aur columns me show karne ke liye <table> tag use hota hai.

  • <table> - Table create karta hai.
  • <tr> - Table row define karta hai.
  • <th> - Table heading define karta hai.
  • <td> - Table data cell define karta hai.

Tables student data, pricing charts aur reports dikhane ke liye use hote hain. 

 
<table> 
  <tr> 
    <th>
      Name
    </th> 
    <th>
      Age
    </th> 
  </tr> 
  <tr> 
    <td>
      Suraj
    </td> 
    <td>
      25
    </td> 
  </tr> 
</table>

Forms

HTML forms user se input lene ke liye use kiye jate hain. Login, signup aur contact forms isi ka example hain.

  • <form> - Form create karta hai.
  • <input> - User input field create karta hai.
  • placeholder - Input field me hint text show karta hai.
  • <button> - Form submit karne ke liye use hota hai.

Forms websites par user interaction ke liye bahut important hote hain. 

 
<form> 
<input type="text" placeholder="Enter Name"> 
<button> 
Submit <
/button> 
</form>

Semantic Tags

Semantic tags webpage ke different sections ka meaning clearly define karte hain. Yeh SEO aur accessibility dono improve karte hain.

  • <header> - Website ka top section define karta hai.
  • <nav> - Navigation links ke liye use hota hai.
  • <main> - Main content area define karta hai.
  • <section> - Different webpage sections banata hai.
  • <article> - Independent content define karta hai.
  • <footer> - Website ka bottom section define karta hai.

Semantic tags clean structure aur better SEO ke liye bahut important hote hain. 


 <header></header> 
 <nav></nav> 
 <main></main> 
 <section></section> 
 <article></article> 
 <footer></footer>
