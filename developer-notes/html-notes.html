HTML Boilerplate

HTML Boilerplate ek basic structure hota hai jo har HTML webpage banane ke liye use kiya jata hai. Isme webpage ke important tags hote hain jaise:

  • <!DOCTYPE html> - Browser ko batata hai ki document HTML5 hai.
  • <html> - Pure webpage ka root element hota hai.
  • <head> - Isme webpage ki information hoti hai jaise title, meta tags, CSS links etc.
  • <meta charset="UTF-8"> - Different languages aur symbols support karta hai.
  • <meta name="viewport"> - Website ko mobile responsive banata hai.
  • <title> - Browser tab me webpage ka naam show karta hai.
  • <body> - Webpage ka visible content yahan likha jata hai.

Har HTML project ki shuruaat isi basic structure se hoti hai. 

    
    <!DOCTYPE html>
    <html lang="en">

      <head>
          <meta charset="UTF-8">
          <meta name="viewport"
                content="width=device-width, initial-scale=1.0">

          <title>Document</title>
      </head>

    <body>

    </body>

    </html>
