Complete CSS3 notes with examples, layouts, animations, flexbox, grid, responsive design and modern UI concepts.

CSS Selectors

CSS Selectors ka use HTML elements ko target karne ke liye hota hai taki un par styling apply ki ja sake.

  • Element Selector - HTML tag name ko target karta hai.
  • Class Selector - Class name ko target karta hai aur . se start hota hai.
  • ID Selector - Unique element ko target karta hai aur # se start hota hai.
  • Selectors webpage ke different elements ko customize karne me help karte hain.

CSS selectors web designing ka basic foundation hote hain aur styling control karne ke liye bahut important hote hain.

Colors

CSS me colors ka use webpage ke text, background aur elements ko attractive banane ke liye kiya jata hai.

  • color - Text ka color change karta hai.
  • background ya background-color - Element ka background color set karta hai.
  • CSS me colors ko name, HEX, RGB aur HSL format me likh sakte hain.
  • Proper color combination webpage ko professional aur readable banata hai.

Web design me colors user experience aur visual appearance ke liye bahut important hote hain. 



/* COLOR USING NAME */
p{
  color:red;
}

/* COLOR USING HEX */
h2{
  color:#7cf03d;
}

/* COLOR USING RGB */
h3{
  color:rgb(124,240,61);
}

/* COLOR USING HSL */
h4{
  color:hsl(96, 86%, 59%);
}

/* PROFESSIONAL COLOR COMBINATION */
.card{
  background:#1f2937;
  color:#ffffff;
  border:1px solid #7cf03d;
}

Box Model

CSS Box Model webpage ke har element ka structure define karta hai. Har HTML element ek box ki tarah behave karta hai.

  • margin - Element ke bahar ki spacing set karta hai.
  • padding - Content aur border ke beech ki spacing set karta hai.
  • border - Element ke around border create karta hai.
  • box-sizing: border-box - Width aur height calculation ko easy banata hai.
  • Box Model webpage layout aur spacing control karne ke liye bahut important hota hai.

Modern websites me layout aur responsive design banane ke liye Box Model ka concept samajhna bahut zaroori hai. 



/* RESET DEFAULT SPACING */

*{
  margin:0;
  padding:0;
  box-sizing:border-box;
}

/* BOX MODEL EXAMPLE */

.box{
  width:300px;

  padding:20px;

  border:5px solid #7cf03d;

  margin:30px;

  background:#1f2937;

  color:white;
}

Flexbox

Flexbox ek powerful CSS layout system hai jo elements ko easily align, distribute aur responsive banane me help karta hai.

  • display: flex - Element ko flex container banata hai.
  • justify-content - Horizontal alignment control karta hai.
  • align-items - Vertical alignment control karta hai.
  • flex-direction - Items ki direction set karta hai.
  • flex-wrap - Items ko next line me wrap karne deta hai.
  • gap - Flex items ke beech spacing deta hai.
  • align-content - Multiple rows ki alignment control karta hai.
  • flex-grow - Extra space occupy karne deta hai.
  • flex-shrink - Element ko shrink hone control karta hai.
  • flex-basis - Initial size define karta hai.

Modern responsive layouts aur professional UI design banane ke liye Flexbox bahut important hota hai. 



/* FLEX CONTAINER */

.container{

  display:flex;

  justify-content:center;

  align-items:center;

  gap:2rem;
}

/* FLEX DIRECTION */

.container{
  flex-direction:row;
}

/* FLEX WRAP */

.container{
  flex-wrap:wrap;
}

/* JUSTIFY CONTENT */

.container{
  justify-content:space-between;
}

/* ALIGN ITEMS */

.container{
  align-items:center;
}

/* ALIGN CONTENT */

.container{
  align-content:center;
}

/* FLEX GROW */

.box{
  flex-grow:1;
}

/* FLEX SHRINK */

.box{
  flex-shrink:1;
}

/* FLEX BASIS */

.box{
  flex-basis:200px;
}

CSS Grid

CSS Grid ek powerful 2D layout system hai jo rows aur columns ki help se responsive layouts banane ke liye use hota hai.

  • display: grid - Element ko grid container banata hai.
  • grid-template-columns - Columns ka size aur number define karta hai.
  • grid-template-rows - Rows ka size define karta hai.
  • repeat() - Repeated values ko short way me likhne ke liye use hota hai.
  • fr - Available space ko equally divide karta hai.
  • gap - Grid items ke beech spacing deta hai.
  • justify-items - Horizontal alignment control karta hai.
  • align-items - Vertical alignment control karta hai.
  • place-items - Horizontal aur vertical alignment ek sath set karta hai.

Modern responsive websites aur dashboard layouts banane ke liye CSS Grid bahut important hota hai. 



/* GRID CONTAINER */

.container{

  display:grid;

  grid-template-columns:
  repeat(3,1fr);

  gap:2rem;
}

/* GRID ROWS */

.container{
  grid-template-rows:
  200px 200px;
}

/* REPEAT FUNCTION */

.container{
  grid-template-columns:
  repeat(4,1fr);
}

/* JUSTIFY ITEMS */

.container{
  justify-items:center;
}

/* ALIGN ITEMS */

.container{
  align-items:center;
}

/* PLACE ITEMS */

.container{
  place-items:center;
}

Position

CSS me position property ka use elements ki location control karne ke liye hota hai.

  • static - Default position hoti hai.
  • relative - Element apni original position ke according move hota hai.
  • absolute - Element nearest positioned parent ke according move hota hai.
  • fixed - Element screen par fixed rehta hai.
  • sticky - Scroll karne par specific position par stick ho jata hai.
  • top, right, bottom, left properties positioning control karti hain.

Navigation bars, floating buttons aur custom layouts banane ke liye position property bahut important hoti hai. 



/* RELATIVE POSITION */

.box{
  position:relative;

  top:20px;

  left:20px;
}

/* ABSOLUTE POSITION */

.box{
  position:absolute;

  top:0;

  left:0;
}

/* FIXED POSITION */

.navbar{
  position:fixed;

  top:0;

  width:100%;
}

/* STICKY POSITION */

.header{
  position:sticky;

  top:0;
}

Animation

CSS Animation ka use webpage elements ko move, fade, rotate aur attractive effects dene ke liye kiya jata hai.

  • animation-name - Animation ka naam define karta hai.
  • animation-duration - Animation kitne time tak chalegi define karta hai.
  • animation-iteration-count - Animation kitni baar chalegi define karta hai.
  • animation-delay - Animation start hone se pehle delay deta hai.
  • animation-direction - Animation ki direction control karta hai.
  • animation-timing-function - Animation ki speed curve control karta hai.
  • @keyframes - Animation ke steps define karta hai.
  • transform - Element ko move, rotate, scale ya skew karta hai.

Modern UI aur interactive websites banane ke liye CSS animations bahut useful hote hain. 



/* BASIC ANIMATION */

.box{

  animation:move 2s infinite;
}

/* KEYFRAMES */

@keyframes move{

  0%{
    transform:translateX(0);
  }

  100%{
    transform:translateX(100px);
  }

}

/* ANIMATION DURATION */

.box{
  animation-duration:2s;
}

/* ANIMATION DELAY */

.box{
  animation-delay:1s;
}

/* ITERATION COUNT */

.box{
  animation-iteration-count:infinite;
}

/* ANIMATION DIRECTION */

.box{
  animation-direction:alternate;
}

/* TIMING FUNCTION */

.box{
  animation-timing-function:ease-in-out;
}

/* ROTATE EXAMPLE */

.box:hover{
  transform:rotate(45deg);
}

/* SCALE EXAMPLE */

.box:hover{
  transform:scale(1.2);
}

Media Query

Media Query ka use responsive website banane ke liye kiya jata hai. Isse different screen sizes ke according styling change ki ja sakti hai.

  • @media - Different devices aur screen sizes ko target karta hai.
  • max-width - Specific width se choti screen par styles apply karta hai.
  • min-width - Specific width se badi screen par styles apply karta hai.
  • Responsive design mobile, tablet aur desktop ke liye important hota hai.
  • Media Queries website ko har device par properly visible banati hain.

Modern web development me responsive design banane ke liye Media Queries bahut important hoti hain. 



/* MOBILE DEVICES */

@media(max-width:768px){

  h1{
    font-size:3rem;
  }

}

/* TABLET DEVICES */

@media(max-width:992px){

  .container{
    flex-direction:column;
  }

}

/* LARGE SCREENS */

@media(min-width:1200px){

  body{
    background:#111;
  }

}

CSS Variables

CSS Variables ka use reusable values store karne ke liye hota hai. Inhe Custom Properties bhi kaha jata hai.

  • :root - Global variables define karne ke liye use hota hai.
  • --variable-name - CSS variable create karta hai.
  • var() - Variable ki value use karne ke liye use hota hai.
  • CSS Variables se code reusable aur maintainable banta hai.
  • Theme colors aur common spacing manage karne ke liye variables bahut useful hote hain.

Modern websites me dark mode, themes aur reusable styling ke liye CSS Variables ka bahut use hota hai. 



/* GLOBAL VARIABLES */

:root{

  --green:#7cf03d;

  --bg-color:#0d1117;

  --text-color:#ffffff;

  --card-bg:#1f2937;
}

/* USING VARIABLES */

body{

  background:var(--bg-color);

  color:var(--text-color);
}

h1{

  color:var(--green);
}

.card{

  background:var(--card-bg);

  border:1px solid var(--green);
}
